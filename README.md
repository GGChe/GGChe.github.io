# GGChe.github.io

Personal portfolio and academic website for Gabriel Galeote-Checa.

Built with the [devfolio](https://github.com/mmacneil/devfolio) template (Bulma CSS, Font Awesome, Flickity).

## Structure

- `index.html` — Main single-page site
- `css/` — Compiled Bulma + custom styles
- `js/` — Bundled JavaScript (modals, carousels)
- `img/` — Images (profile, portfolio, carousel)
- `files/` — Downloadable publication PDFs
- `docs/` — Resume file
- `src/` — Webpack/SCSS source for development

## Development

To rebuild CSS/JS from source:

```bash
cd src
npm install
npm run build
```
